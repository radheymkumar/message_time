MESSAGE TIME

INTRODUCTION
------------
	This module using for auto hide message


INSTALLATION
------------


CONFIGURATION
-------------
	Goto: /admin/config/system/site-information
	- Message Time
		Now the message box will disappear after 10 seconds, as we hardcoded the duration (10000ms = 10 sec)
